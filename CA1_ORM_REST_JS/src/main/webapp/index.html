On this site you can easily create, read, update and delete information about specific objects

After you have chosen a subject you can edit them

This REST API can get a person or a company from a database.
It can also insert into the database via POST.
All results are in JSON format.
All Commands are prefetched with: http://dev.breindal.me/CA1_ORM_REST_JS/api/
commands include for persons:
/person/complete - returns all persons in database.
/person/complete/id - substitute ID with the ID of the person you want. Persons are ID 51-100.
/person/contactinfo - returns all persons, but only their contactinfo.
/person/contactinfo/id -substitute ID with id of a person you want the contactinfo off.
/person - POST request - send a post request and add the person in json format to the database.
/person/id - delete the person with the specific id.
Commands for Company:
/company/complete - get all companies with all info.
/company/complete/id - sub id for id of company. IDs for company are 1-50.
/company/id - delete company with this id.
/comany - POST - Add the company in json to the database.
